# Water Timer 💧

> 작업 중에 깜박하고 물 안 마시는 사람을 위한, 조용하고 귀여운 Windows 리마인더.

설정한 간격마다 화면 구석에 작은 팝업이 떠서 **물방울 캐릭터**가 물 한 잔 어때요 하고 말을 걸어줍니다. "마셨어요!" 버튼 한 번 누르면 오늘 마신 잔 수가 올라가고, 하루 목표까지 얼마나 남았는지 한눈에 보여요.

---

## 📥 설치하기

1. [**Releases 페이지**](../../releases/latest) 에서 `WaterTimer-Setup-0.1.0.exe` 를 다운로드하세요.
2. 다운로드한 파일을 더블클릭.
3. Windows 가 `PC를 보호했습니다` 파란 창을 띄우면 **"추가 정보" → "실행"** 눌러주세요. 처음 한 번만 뜨는 보안 경고예요.
4. 설치 마법사에서 **다음 → 설치 → 마침**. 끝.

설치가 끝나면 작업표시줄 오른쪽 아래(시계 옆)에 파란 **물방울 아이콘**이 뜹니다. 아이콘이 안 보이면 `∧` 화살표 눌러서 숨겨진 영역을 열어보세요.

---

## 🚀 처음 쓰기

**아이콘을 더블클릭**하면 설정 창이 열립니다. 탭이 5개 있어요:

| 탭 | 뭘 할 수 있나요 |
|---|---|
| 🔔 **알림** | 몇 분마다 알림 뜰지, 몇 시부터 몇 시까지 울릴지, 어느 요일에만 울릴지, 팝업 위치(화면 어느 구석) |
| 📊 **기록** | 오늘 몇 잔 마셨는지, 주간 그래프, 주 평균/연속 달성 통계. 하루 목표(기본 8잔) 조정도 여기서 |
| 🎨 **커스터마이즈** | 팝업에 나올 **캐릭터 선택** (기본 3종 + 내 이미지 여러 장 올려서 랜덤 순환), 알림 메시지 문구 커스텀 |
| 🔊 **사운드** | 알림음 on/off, 볼륨, 소리 종류 5개 중 선택 |
| ⚙️ **시작·트레이** | Windows 시작 시 자동 실행, 트레이 아이콘 닫을 때 동작 |

### 팝업 동작

팝업이 뜨면 세 가지 버튼이 있어요:
- **마셨어요!** — 오늘 카운터 +1, 물방울 파티클 튀면서 닫힘
- **5분 뒤** — 지금은 바쁘니 5분 뒤 다시 알려줘
- **×** — 그냥 닫기 (카운터 변화 없음)

그냥 놔두면 12초 뒤 자동으로 사라집니다 (자동 닫힘 시간은 설정에서 조정).

### 캐릭터 여러 개 쓰기

🎨 **커스터마이즈** 탭에서:
1. "**+ 업로드**" 카드 눌러서 원하는 이미지 여러 장 추가 (내 반려동물 사진, 친구 사진, 좋아하는 밈 등 뭐든 OK)
2. 쓰고 싶은 이미지·캐릭터 카드를 각각 클릭해 파란 테두리로 **선택**
3. 알림이 뜰 때마다 선택한 것들 중 **랜덤**으로 나옴

업데이트 후 업그레이드된 기능이 반영되려면 트레이에서 앱을 한 번 종료했다가 다시 열어주세요.

## 🛠️ 문제가 있을 때

### 알림이 안 떠요 / 이상하게 동작해요

`Win + R` 누르고 `%APPDATA%\WaterTimer` 입력 → Enter. 폴더가 열리면 `error.log` 파일을 배포자에게 보내주세요.

설정 파일이 꼬였다 싶으면 같은 폴더의 `config.json` 을 지우고 앱을 재시작하면 기본값으로 돌아옵니다 (업로드한 이미지는 `characters\` 폴더에 별도 저장).

### 자동 실행이 안 돼요

설정 → **⚙️ 시작·트레이** → "Windows 시작 시 자동 실행" 토글을 한 번 껐다 켜보세요. 레지스트리 항목이 현재 설치 경로로 갱신됩니다.

### 앱을 완전히 제거하고 싶어요

Windows **제어판 → 프로그램 추가/제거** 에서 "Water Timer" 찾아 제거. 사용자 데이터(설정·기록·업로드 이미지)는 `%APPDATA%\WaterTimer\` 에 남아있으니, 완전 삭제를 원하면 그 폴더도 직접 삭제하세요.

## 💻 개발자용 정보

소스를 받아 직접 빌드하거나 수정하고 싶으면:

```
git clone https://github.com/<owner>/water-timer.git
cd water-timer
py -3.12 -m venv .venv
.venv\Scripts\activate
pip install -r requirements-dev.txt

# 개발 실행
python -m src.app

# 테스트
pytest tests/ -v

# 배포용 빌드 (.exe + 인스톨러)
build.bat
```

배포 전체 흐름은 [`DEPLOY.md`](./DEPLOY.md) 참조.
